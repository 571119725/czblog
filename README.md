## 使用方法

1. 运行`./init.sh`,下载二进制(需要github连接,这一步可以自行下载)

2. 运行`./update.sh`,生成book文件夹

3. 进入`./backend`,执行`make`生成`./gowebquick`二进制文件,运行它

4. 打开浏览器`http://127.0.0.1:5203`就可以看到效果
